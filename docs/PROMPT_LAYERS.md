# 提示詞與本機判定

主解題只接收必要內容：固定 JSON／排版規格、題目、使用者勾選章節的解題提醒，以及題型判讀 AI 選出的短通則卡條件。

```text
題型判讀 AI（只依題意選卡）
  → 主解題：固定骨架 + 題目 + 章節提醒（若有）+ 短判定條件（若有）
  → 本機驗收可確認的化學條件
```

## 固定骨架

- 每題：題意 → 依據與推導 → 結果。
- 選擇題最後加選項分析，逐項對應題目選項。
- 純代數可精簡；選用判準、轉換化學意義與排除選項時必須說明理由。

## 通則卡

- 題型判讀 AI 只負責依題意選擇最適用的一張卡，不寫詳解或答案。
- 卡片注入內容只保留適用的判定條件與「題幹反應式、計量關係、實驗條件優先」原則。
- 碘鐘題若本機可由題幹完整資料確認 `n(IO3-)/n(HSO3-) ≤ 1/3`，本機驗收不變藍與時間不適用；不完整資料不作硬判。

JSON schema、文字與算式的顯示規則由 `js/solution-core.js` 和本機編譯器固定，不以提示詞的優先宣告處理。
