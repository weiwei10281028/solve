【GitHub Pages 發布 — 3 步驟】

你的程式在 GitHub 上是這樣放的：
  solve（儲存庫）
    └── AI解題（資料夾）
          └── index.html …

請把「本資料夾裡的 .github」上傳到 solve 的最外層（和 AI解題 同一層）。

上傳後結構應為：
  solve
    ├── .github/workflows/pages.yml   ← 從這裡複製上去
    ├── AI解題
    │     ├── index.html
    │     └── …
    └── （其他檔案）

然後到 GitHub 網頁：
1. 打開 solve 專案 → Settings → Pages
2. Build and deployment → Source 選「GitHub Actions」
3. 等 1～3 分鐘，Actions 跑完後會出現網址

你的解題網頁網址（完成後）：
  https://weiwei10281028.github.io/solve/

題庫匯入：
  https://weiwei10281028.github.io/solve/db-import.html

【重要】私人（Private）儲存庫可能無法免費用 Pages。
若 Settings → Pages 沒有網址，請把 solve 改成 Public：
  Settings → General → Danger Zone → Change visibility → Public

API 金鑰仍只存在各自瀏覽器，不會上傳到 GitHub。
