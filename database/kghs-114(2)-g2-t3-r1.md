### 類題｜反應速率同型（速率定律／係數和／級數）
<!-- MATCH: 反應速率,速率定律,速率常數,反應級數,半衰期,碰撞學說,活化能,Δn氣體係數和,總壓不變,濃度時間圖,零級一級二級,速率比係數比 -->

速率定律：
$$r = k[A]^m[B]^n$$

氣體反應（定溫定容）係數和判斷：
$$\Delta n = \sum \nu_{\text{產物,g}} - \sum \nu_{\text{反應物,g}}$$
$$\Delta n = 0 \implies \text{總壓不變，不宜用總壓測速率}$$

速率常數單位與級數：
$$k \text{ 之單位為 } M^{1-n}\cdot\text{time}^{-1} \implies n = \htmlData{note=反應級數}{n}$$

### 第 1 題

<!-- MATCH: N2O4分解2NO2,NO2紅棕色顏色變化,H2Cl2定容總壓,H2Cl2氣體係數和相同,碳酸鈣CO2分壓,N2H2定壓體積變化,反應速率測定方法,不適當的測速率方式 -->

(A) $NO_2$ 為紅棕色氣體，可利用顏色變化判斷速率。

(B) $H_{2(g)} + Cl_{2(g)} \rightarrow 2HCl_{(g)}$，反應物與生成物氣體係數和皆為 $2$，定溫定容下總壓不變，無法用總壓變化判斷速率。

(C) $CaCO_{3(s)} \rightarrow CaO_{(s)} + CO_{2(g)}$，反應後 $CO_2$ 莫耳數增加，$CO_2$ 分壓變大，可利用 $CO_2$ 分壓變化判斷速率。

(D) $N_{2(g)} + 3H_{2(g)} \rightarrow 2NH_{3(g)}$，反應物與生成物氣體係數和不同，定溫定壓下體積會變，可利用體積變化判斷速率。

### 第 2 題

<!-- MATCH: 定溫定容總壓變化,Δn氣體莫耳數變化,氣體係數和相同總壓不變,H2O2生成液態,H2Br2生成2HBr,HBr氧化,PH3分解,N2O4分解,不能用總壓測速率 -->

定溫定容下，總壓變化 $\\Delta P \\propto \\Delta n_{gas}$。
$\\Delta n = \\text{生成物氣體係數和} - \\text{反應物氣體係數和}$；
$\\Delta n > 0$ 總壓變大，$\\Delta n < 0$ 總壓變小，$\\Delta n = 0$ 總壓不變。

(A) $2H_{2(g)} + O_{2(g)} \rightarrow 2H_2O_{(l)}$，$\\Delta n = 0 - 3 = -3 \\neq 0$，可用總壓變化測定速率。

(B) $4PH_{3(g)} \rightarrow P_{4(g)} + 6H_{2(g)}$，$\\Delta n = 7 - 4 = 3 \\neq 0$，可用總壓變化測定速率。

(C) $4HBr_{(g)} + O_{2(g)} \rightarrow 2Br_{2(g)} + 2H_2O_{(g)}$，$\\Delta n = 4 - 5 = -1 \\neq 0$，可用總壓變化測定速率。

(D) $N_2O_{4(g)} \rightarrow 2NO_{2(g)}$，$\\Delta n = 2 - 1 = 1 \\neq 0$，可用總壓變化測定速率。

(E) $H_{2(g)} + Br_{2(g)} \rightarrow 2HBr_{(g)}$，$\\Delta n = 2 - 2 = 0$，總壓不變，無法利用總壓變化測定反應速率。

$2O_{3(g)} \rightarrow 3O_{2(g)}$

$$R_{O_3} : R_{O_2} = 2 : 3 = 6.6 \\times 10^{-3} : x$$

### 第 4 題

<!-- MATCH: 速率常數單位,L mol-1 min-1,M的-1次方,二級反應判定,n級反應速率常數單位,2.55e-2 -->

<!-- MATCH: 5Br-加BrO3-加H+,速率定律r=kBr-BrO3,混合100mL1MKBr,0.1MKBrO3,6MH2SO4,剩餘BrO3-0.03M,反應速率比值,濃度變化表 -->

$$\\begin{array}{c|cc}

\\end{array}$$

$$\\dfrac{R}{s} = \\dfrac{\\htmlData{note=剩餘Br-濃度}{0.2} \\times \\htmlData{note=剩餘BrO3-濃度}{0.03}}{0.2 \\times 0.02} = \\dfrac{0.006}{0.004} = 1.5$$

若題幹以初濃 $0.5\\,M$、$0.05\\,M$ 計算（詳解板書）：$\\dfrac{R}{s} = \\dfrac{0.4 \\times 0.03}{0.5 \\times 0.05} = 0.48$，$R = 0.48s$。

### 第 10 題

### 第 11 題

<!-- MATCH: 2A生成B加C,一級反應等比遞減,平均速率法求k,半生期0.693除以k,濃度表1.0 0.8 0.64 -->

$2A \\rightarrow B + C$，時間等間隔，$[A]$：$1.0 \\rightarrow 0.8 \\rightarrow 0.64\\,M$（$0,20,40\\,\\text{min}$），濃度等比遞減，為一級反應。

平均速率法：
$$\\dfrac{1.0 - 0.8}{20 - 0} = k \\times \\left(\\dfrac{1.0 + 0.8}{2}\\right)$$

$$k = \\htmlData{note=速率常數}{\\dfrac{1}{90}}\\,\\text{s}^{-1}$$

$$t_{1/2} = \\dfrac{0.693}{k} = \\dfrac{0.693}{1/90} = \\htmlData{note=半生期}{62.37}\\,\\text{s}$$

### 第 13 題

<!-- MATCH: 不可逆反應,濃度時間關係圖,0.2M到0.1M四分鐘,0.1M到0.05M八分鐘,半生期逐次加倍,二級反應 -->

### 第 14 題

<!-- MATCH: 碰撞學說,活化能活化複合體,有效碰撞方向,低限能粒子數,一級反應速率與濃度一次方 -->

(A) 反應物粒子互相碰撞，需有足夠能量與正確碰撞方向才會發生反應。

(B) 活化能為活化複合體與反應物（或生成物）的熱含量差。

(C) 活化能越大，超過低限能的粒子數越少，反應速率越慢。

(D) 若為一級反應，反應速率才與反應物濃度成正比（$1$ 次方關係）。

### 第 15 題

欲形成新鍵結，須使成鍵原子互相接觸；有效碰撞位向為 $CO$ 的 $C$ 原子撞擊 $NO_2$ 的 $O$ 原子。

### 單選 第 16 題
<!-- MATCH: 反應機構三步驟,A+B生成C慢,C+2A生成D+E,E生成F,中間物C與E,瓶頸反應,淨反應3A+B生成D+F,速率定律R=k[A][B] -->

機構：(1) $A+B \rightarrow C$（慢）；(2) $C+2A \rightarrow D+E$（快）；(3) $E \rightarrow F$（快）。

(A) 第一步最慢，速率定律式 $R = k[A][B]$。
(B) 淨反應：$3A + B \rightarrow D + F$。
(C) $C$、$E$ 為中間物。
(D) 第 (1) 步為瓶頸反應（速率決定步驟）。

### 多選 第 1 題
<!-- MATCH: 反應速率測定,N2與O2生成NO無色,顏色變化,H2與Cl2總壓不變,CaCO3二氧化碳分壓,NH3合成總質量不變,H2O分解體積變化 -->

(A) $N_2 + O_2 \rightarrow 2NO$，三者皆無色，不能以顏色變化測速率。
(B) $H_2 + Cl_2 \rightarrow 2HCl$，氣體係數和相同，總壓不變。
(C) $CaCO_3 \rightarrow CaO + CO_2$，$CO_2$ 分壓增大，可測速率。
(D) $N_2 + 3H_2 \rightarrow 2NH_3$，氣體總質量不變，不能以總質量變化測速率。
