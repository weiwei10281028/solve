## 教學規定

【何時套用】題目須**判斷混成軌域種類**（含選項比較、中心原子混成、混成與分子形狀連結）時使用。僅問**原子**基態電子組態、光譜、游離能等、**不涉及鍵結與混成**者，不套用。

【板書原則】詳解**每題只選一法**推到底；**禁止**把 (a)(b)(c) 三法全文或講義標題貼進正文。流程：標**中心原子** → 用該法計數 → 得**混成種類** → 必要時補 **VSEPR 形狀** → 再評析 (A)～(E)。

【方法選擇（優先順序）】
1. **(a) 法（預設）**：一般無機、離子、配位化合物；能判中心原子之 σ 鍵數與 lp 時。
2. **(b) 法**：**有機分子**、中心原子**符合八隅體**且 π 鍵判斷清楚時。**NO₂ 為例外**，仍用 (a) 法。
3. **(c) 法**：**僅一個中心**（整分子或離子只有一個待判中心）、總價電子為**偶數**、欲由價電子快速得 σ 與 lp 時。**不可用於多中心各別混成**（如醋酸兩個 C 須分別判）。

---

### (a) 法：σ 鍵數＋孤對電子對數 (lp)

**邏輯：** 中心原子混成軌域數 ＝ 與中心相連之 **σ 鍵總數** ＋ 中心之 **孤對電子對數 (lp)**。（π 鍵不另占混成軌域；lp 計**對數**，非電子個數。）

**對照：** 2→sp；3→sp²；4→sp³；5→sp³d；6→sp³d²

**範例（內部參考）：**
- $BF_3$：中心 B，3 σ、0 lp → 3 → sp²（平面三角）
- $I_3^-$：中心 I，2 σ、3 lp → 5 → sp³d（直線型）
- $N_2H_4$：中心 N，3 σ、1 lp → 4 → sp³（三角錐）

---

### (b) 法：π 鍵數

**適用：** 有機物為主；中心原子**符合八隅體**（**NO₂ 除外**）。

**邏輯：** 中心原子 π 鍵數 → 0 個 π→sp³；1 個 π→sp²；2 個 π→sp。（n 個 π 表示 n 個 p 軌域未混成。）

**範例：**
- $H_2S$：中心 S，0 π → sp³（角形）
- $N_2O$（中心 N）：2 π → sp（直線型）
- $CH_3COOH$：甲基 C 0 π→sp³；羧基 C 1 π→sp²（**兩中心各用 (b) 法，不可用 (c) 法一次算完**）
- **$NO_2$（例外）**：中心 N 有 2 σ、1 孤電子（占 1 混成軌域）→ 共 3 → **sp²**（角形）；**禁止**用 2 個 π 直接判 sp

---

### (c) 法：價電子計算

**限制：**
- 僅**單一中心**分子或離子
- **H 原子**計算時視為 **7 個價電子**
- 總價電子為**奇數**者**不可用**
- 多中心分子（如 $CH_3COOH$）**各中心分別計算**，不可用總價電子一次定全分子混成

**步驟：**
1. 加總價電子（H 各 7；離子加減電荷）
2. 總價電子 ÷ 8 → 商 **A**（σ 鍵數），取餘數
3. 餘數 ÷ 2 → 商 **B**（lp 對數）
4. 混成軌域數 ＝ **A ＋ B** → 對照 (a) 法表

**範例：**
- $SO_3^{2-}$：26 → 26÷8＝3 餘 2；2÷2＝1 → 3+1＝4 → **sp³**（三角錐）
- $H_2O$：6+7×2＝20 → 2 餘 4；4÷2＝2 → 2+2＝4 → **sp³**（角形）
- $XeF_4$：36 → 4 餘 4；4÷2＝2 → 4+2＝6 → **sp³d²**（四方平面）

【板書示範（單中心用 (a) 法，勿照抄標題）】
中心 Sb（例）：σ＋lp＝… → 混成 sp³d²；再寫形狀；接 (A)～(E)。

【禁止】
- 正文列出三法教學或「判斷混成軌域種類的方法」標題
- 同一中心混用三法得不同答案而不說明
- NO₂ 用 (b) 法 π＝2 判 sp
- (c) 法用於奇數價電子或多中心未分開計算
- **追問與詳解均禁止** VSEPR 之 $V+X-q$（或 $V+M-q$）、禁止總價電子直接 $\\div 2$ 當混成軌域數（須 (c) 法先 $\\div 8$ 再餘數 $\\div 2$）
