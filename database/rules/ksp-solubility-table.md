## 教學規定

【何時套用】題目須以 **$K_{sp}$／溶解平衡** 求離子殘量、溶解度，或 **兩溶液混合後** 判斷是否沉澱並再求平衡濃度時使用。**觀念敘述題**（何者正確、沸騰汽化蒸氣壓等，無計算求值）**不套用**。

【反應式】難溶鹽寫在 **左側**，須出現在變化表 **第一列**（與資料列同一 `array`），例如第一列 $AgCl_{(s)} \rightleftharpoons Ag^+ + Cl^-$，再接 $\text{起始}$、$\text{完全向左}$ 等列。**禁止**淨離子單向式當主表；**禁止**只寫「反應式如下：」卻不輸出 `array`。

固相在表內以 $(s)$ 標示；水溶液離子勿加 $(aq)$ 亦可。

【變化表（四列，預設）】混合後須先 **完全沉澱／完全向左**，再談 **再溶解（向右 $x$）** 與平衡時，板書須用下列結構（標籤逐字使用）：

| 列標籤 | 內容要點 |
|--------|----------|
| $\text{起始}$ | 抄題幹混合後、沉澱發生前之各物種莫耳數或濃度（**禁占位 1、2**） |
| $\text{完全向左}$ | **變化量**（非結果）：固相 **+$s$**、限量離子 **$-\text{限量}$**、另一離子依係數 **$-\text{消耗量}$**（例：$+0.020,\,-0.020,\,-0.020$） |
| $\text{再向右}$ | 固相再溶解之變化量（例：$-x,\,+x,\,+x$） |
| $\text{平衡}$ | **最終平衡量**＝起始＋各變化列相加（例：$0.020-x,\,x,\,0.080+x$）；末列亦可寫 $\text{結果}$ |

**單步限量、無再溶解**者，可省略「完全向左／再向右」，改三列：$\text{起始}$ → $\text{變化}$ → $\text{結果}$。

【計算】平衡後代入
$$K_{sp} = [Ag^+][Cl^-]$$
或題目給定之離子積式。小量可加性可寫 $x(0.08+x) \approx 0.08x$（須註明小量省略理由）；**結果須化簡**，勿留 $1-0.020$ 未算式。

【板書順序】反應式如下：→ `array` 表 → 寫 $K_{sp}$ 等號式 → 解 $x$ → 再評析選項。

【禁止】
- 淨離子單向式當主表反應式
- 表內占位數 1、2、0 未由題意推出
- 把本段標題或「教學規定」字樣貼入詳解正文
